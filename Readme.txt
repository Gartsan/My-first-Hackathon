# Eric Gichuki Gartsan — Portfolio Website

Welcome to the source code of my personal portfolio website. This is a fully responsive, modern, and professional site that highlights my skills, education, services, and contact details as a junior full-stack software engineer.

## 🚀 Features

- Fully responsive and mobile-friendly design
- Clean hero section with typing animation
- About section with profile and personal bio
- Resume section with education timeline
- Animated skill bars
- Services offered with icon blocks
- Scroll-to-top and preloader features
- Contact section with clickable phone and email
- Social links to LinkedIn and X (Twitter)

---

## 🛠️ Built With

- HTML5  
- CSS3 & Bootstrap 5  
- JavaScript  
- AOS (Animate On Scroll)  
- Bootstrap Icons
